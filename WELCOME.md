# Welcome to Caia Coder

Welcome to the official **community repository** for **Caia Coder**.

**Caia Coder is a subscription product.** Subscribers use the hosted app and [account portal](https://account.coder.caiaplatform.com); they do not clone this repo to run the product.

**This GitHub repository is where you:**

1. **Read** company and user documentation  
2. **Follow** the public [changelog](https://github.com/CaiaSoftware-Studios-LLP/Coder/blob/main/CHANGELOG.md)  
3. **Discuss** how-tos and ideas in [Discussions](https://github.com/CaiaSoftware-Studios-LLP/Coder/discussions)  
4. **Open** [Issues](https://github.com/CaiaSoftware-Studios-LLP/Coder/issues) for bugs and feature requests  

Application source code is private and is not published here.
