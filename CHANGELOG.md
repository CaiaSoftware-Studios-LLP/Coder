# Changelog

**Caia Coder** is a **subscription product**. This file is the **public release history** for subscribers and operators.

- **Product changes** (features, fixes, integrations) are listed below when we publish a release.
- **Application source code** is not in this GitHub repository.
- **Questions or ideas** → [GitHub Discussions](https://github.com/CaiaSoftware-Studios-LLP/Coder/discussions). **Bugs** → [GitHub Issues](https://github.com/CaiaSoftware-Studios-LLP/Coder/issues).

Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2024-05-23

### Added
- Initial release of Caia Coder public documentation.
- Added integration guides for Confluence, Google, observability, Kubernetes, Docker, Figma, SAP, JFrog, GitHub, and PostgreSQL.
- Documented IDE features, marketplace, local vault, and SRE scheduler.
- Restored and published Jekyll user docs under `docs/` (getting started, billing, models, deployment, troubleshooting, audience guide).

## How to stay updated

- Watch this repository on GitHub for release notifications.
- Read [GitHub Discussions](https://github.com/CaiaSoftware-Studios-LLP/Coder/discussions) for announcements.
- Check in-app banners after sign-in for urgent account or billing notices.
